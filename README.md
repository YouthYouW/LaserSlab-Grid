LaserSlab-grid is the setup folder

use command below to setup

./setup -auto LaserSlab -2d +cylindrical -nxb=16 -nyb=16 +hdf5typeio species=cham,targ +mtmmmt +laser +uhd3t +mgd mgd_meshgroups=6 -objdir=cases/LaserSlab -parfile=example.par

generate_flash_par.py is to generate the part of "refine region" that we add in the project. Just duplicate the content in the .txt file generated by .py to the .par file.

refinement_region.csv is to define the regions as what you want and is used by generate_flash_par.py
